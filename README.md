### 🔐 [TrustNet AI](https://github.com/RafayImraan/trustnet)
B2B trust coordination platform for marketplaces, wallets, fintech apps, DAOs, and aid distribution systems. Multi-agent AI pipeline (Fraud → Validation → Trust → Coordinator), anchored on Hedera with immutable HCS audit trails and TRUST token incentives. Multi-tenant with workspace switching and role-aware permissions.
`Node.js` `React` `Hedera` `MongoDB` `Multi-agent AI`

### ⚡ [BSwap](https://github.com/RafayImraan/BSwap)
Trustless cross-chain atomic swap between Bitcoin Testnet and Hedera Testnet. Same hashlock on both chains, enforces `T1 > T2` timelock ordering, non-custodial coordination server. No bridge, no custodian — pure HTLC cryptography.
`JavaScript` `Bitcoin` `Hedera` `HTLCs` `DeFi`
